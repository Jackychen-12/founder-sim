# 创业模拟器 · Founder Sim

> 一个想法，一点点钱，和一段没人能替你走的路。
> 一款纯文字的创业人生模拟游戏 —— 走到终局，它会告诉你：**你究竟是哪一种创始人。**

🔗 **在线试玩：https://jackychen-12.github.io/founder-sim/**

手机、电脑均可直接打开，无需安装。

---

## 这是什么

Founder Sim 不是一款关于胜负的游戏，而是一面镜子。

你选定一条创业赛道，给自己和公司起个名字，然后在 30+ 个真实的创业决策里一次次选择 —— 融资、招人、产品、危机、公关、家庭、健康、学习。没有撤销，没有读档。每一个选择都会改变公司的轨迹，也会被悄悄记下，作为你性格的一份证据。

剧本走到尽头（IPO、被收购、独角兽、资金链断裂、健康崩塌……），游戏会把这些选择拼回成一份关于**你**的画像：你的风险偏好、你对控制与信任的态度、你看时间的长度、你对人的关怀、你向自己学习的能力 —— 并据此推导出你的创始人原型与方向性建议。

---

## 玩法

1. **选赛道** —— 6 条方向，开局数值、专属事件、退出路径各不相同。
2. **多轮经营** —— 每个月遇到一个事件，做出选择，实时影响现金、跑道、团队、健康等数值。
3. **看结局** —— 触发某种结局后，进入「创业人格揭晓」：原型卡 + 五维画像 + 强项/盲区 + 方向建议 + 能力雷达 + 关键抉择回放。

### 你要盯紧的数值

| 维度 | 说明 |
|------|------|
| 💰 现金 / 跑道 | 烧钱速度决定你还能撑几个月，归零即清算 |
| 📈 月收入 MRR | 产品和市场到位后才会自然增长 |
| 🧩 产品 / 技术 / 市场 / 营销 / 品牌 / 团队 | 六项公司能力 |
| 🏠 个人健康 / 📚 认知学习 | 容易被忽略，却最先出问题的两项 |

## 五维性格 & 八种原型

游戏不靠问卷，而是从你的**真实决策**里累积出 5 个创业语境下最难掩饰的人格维度：

- **风险偏好** —— 保守稳健 ↔ 激进押注
- **控制 / 分权** —— 分权信任 ↔ 紧握权柄
- **时间视角** —— 短期套现 ↔ 长期主义
- **人本关怀** —— 工具理性 ↔ 人本关怀
- **认知开放** —— 封闭执行 ↔ 反思学习

由这 5 维组合，匹配出最接近你的一种创始人原型（共 8 种）：

> 务实合伙型 · 激进孤狼型 · 反思型工匠 · 远见操盘手 · 稳健守业者 · 野心野战派 · 温柔理想家 · 学习型新手

每种原型都附带：一句话画像、你最亮的地方、最容易绊倒你的盲区、适合 / 应避免的赛道，以及一份基于你这一局决策生成的**方向性建议**。

## 特性

- 📄 **单文件 · 离线运行** —— 整个游戏就是一个 `index.html`，无依赖、无后端、无网络请求。
- 🎨 **纸质杂志 / 心理报告风** —— 暖米白纸张底 + 衬线体 + 墨绿 / 赭石点缀。
- 📱 **移动端优先** —— 顶部紧凑 HUD + 底部状态抽屉，安全区适配，大号可点选项；桌面端自动切换为两栏（剧情 + 常驻状态栏）。
- ⌨️ **沉浸交互** —— 对话逐字打字机、数值浮动反馈、阶段里程碑提示、轻震动、可选音效、场景转场、结局逐段揭晓。
- 🔒 **隐私无忧** —— 所有数据仅存于浏览器内存，关闭页面即清除，不上传、不存储。

## 技术说明

- 纯原生 **HTML + CSS + JavaScript**，零第三方依赖。
- 剧情引擎：事件库 `EVENTS` + 赛道定义 `DIRECTIONS` + 按月份 / 阶段 / 条件触发的事件挑选器。
- 性格系统：`ACTION_TRAITS` 把每个选项映射到 5 维分值，累计归一后匹配 `ARCHETYPES` 原型并生成建议。
- 能力雷达用内联 SVG 绘制。
- 部署：GitHub Pages（`main` 分支根目录）。
